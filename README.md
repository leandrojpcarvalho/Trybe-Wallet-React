# Trybe Wallet

## Descrição 
Este aplicativo React é um sistema de gerenciamento de despesas com funcionalidades de login, visualização, adição, edição e exclusão de despesas.

Ele possui duas páginas acessíveis por meio de roteamento: uma para o login, onde os usuários podem inserir email e senha com validação em tempo real, e outra para a carteira, onde as despesas são exibidas em uma tabela. Os usuários podem adicionar novas despesas por meio de um formulário, editar ou excluir despesas individualmente. O valor total das despesas em BRL é calculado com base nas taxas de câmbio e exibido no cabeçalho da página.

Algumas funcionalidades, como a manipulação de estados de formulários, uso do Redux para armazenar dados e compartilhamento de informações de formulário entre componentes por meio do contexto "Test", são compartilhadas entre as páginas. Este aplicativo oferece aos usuários uma maneira organizada de gerenciar suas despesas com validações em tempo real e armazenamento persistente no Redux.
